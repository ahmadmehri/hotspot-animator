Hotspot Animator - Windows Standalone

How to run:
1. Extract the ZIP file.
2. Double-click "Hotspot Animator.exe".
3. Click "Import Image" and choose a hotspot image.
4. Choose Compose layers or Sequence mode.
5. Choose an animation style and adjust settings.
6. Optional: use the Layers/Frames panel for order, visibility, size, position,
   and sequence timing.
7. Optional: click "Output Folder" to save exports to a chosen folder.
8. Click "Export APNG".

Notes:
- Python is not required.
- The app runs locally on Windows.
- The selected output folder is used for APNG files and batch ZIP exports.
- If Windows SmartScreen appears, click "More info" and then "Run anyway".
- Exported APNG files can be used as animated hotspot graphics in 3DVista.
- The About section includes Rock Bench links for VT education, the playlist,
  GitHub, and donation page.
